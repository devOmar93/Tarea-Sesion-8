# One Piece

One Piece es una serie de manga y anime creada por Eiichiro Oda, centrada en las aventuras de Monkey D. Luffy, un joven que obtiene poderes elásticos tras comer una Fruta del Diablo. Inspirado por la leyenda del mítico pirata Gol D. Roger, Luffy emprende un viaje por los mares para encontrar el tesoro legendario conocido como One Piece y convertirse en el Rey de los Piratas.
